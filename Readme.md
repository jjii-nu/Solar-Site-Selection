# 🌞 Korea PV Potential Analysis Project  
### _Photovoltaic Power Generation Forecasting & Site Suitability Analysis in South Korea_

---

## 📌 Project Overview  
본 프로젝트는 **한국 전역의 태양광 발전 잠재력(PV Potential)을 공간적으로 평가하고**,  
**미래 기후 시나리오**를 이용해  
**태양광 발전량 변화(P50·P95)와 최적 입지 후보지를 분석**하는 연구입니다.

해당 연구는 멕시코에서 수행된  
> **"Photovoltaic Power Generation Forecasting for Regional Assessment Using Machine Learning"**  
방법론을 기반으로, **한국 기상 데이터(KMA/ERA5)와 한국형 GCM(KACE)** 등의 자료를 활용하여  
한국 상황에 맞게 재구성한 프로젝트입니다.

---

## 🎯 Research Goal  
### **한국의 태양광 발전 최적지 선정 및 기후 시나리오 기반 발전량 예측**

1. **격자(Grid) 기반 공간 분할**
2. **기상 데이터 수집 (일사량·기온 / 2010–2023)**
3. **태양에너지 + 온도 영향 분석**
4. **PV 모듈 효율·온도 보정 기반 발전량 계산**
5. **SSP 시나리오별 발전량 변화 예측**
7. **미래 태양광 최적 입지 후보지 도출**
